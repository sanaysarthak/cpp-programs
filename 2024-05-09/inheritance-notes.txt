inheritance is the process, by which a class can acquire(reuse) the properties and methods of the parent class
    
base class is also known as parent class or super-class
derived class is also known as child class or sub-class

difference between access modifier is important
in exam, question on different types of inheritance can be asked, answer with diagram and code.

/* Syntax:
class derived-class-name : access-modifier base-class-name
{
    // body of class
};
*/

/* Example:
class person {
    // body of class
};
class doctor : protected person {
    // body of class
};
*/
